# Trekking Fuera de Sendero - San Luis

🏔️ Bienvenido a Trekking San Luis

Un sitio web dedicado a expediciones de trekking fuera de los senderos tradicionales en la provincia de San Luis, Argentina.

## 📋 Características

- Información sobre rutas de trekking
- Guías de aventura
- Recursos para exploradores
- Contacto y reservas

## 🚀 Cómo empezar

1. Clona el repositorio:
   ```bash
   git clone https://github.com/meliflua/trekksanluis.git
   cd trekksanluis
   ```

2. Abre `index.html` en tu navegador

## 📁 Estructura del proyecto

```
trekksanluis/
├── index.html          # Página principal
├── styles/             # Estilos CSS
├── scripts/            # JavaScript
└── images/             # Imágenes y recursos
```
